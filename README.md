# trabQualSoft_pyTest
Trabalho final da disciplina de Qualidade de Software, realizada no quinto semestre da faculdade de 'Sistemas de Informação', na UFSM. O trabalho consiste em realizar teste automáticos utilizando Pytest e Unittest sob um sistema em python de truco gaúcho.

Todos os arquivos fora da pasta '/tests' NÃO SÃO de nossa autoria, e foram fornecidos pelo professor para criarmos testes em cima deles, sem poder alterar o código fonte.
